# История описания документа

Функция позволяет просматривать историю изменений **описания** (description) документа, сравнивать версии и восстанавливать предыдущие.

## Как открыть

- В редакторе Markdown — кнопка **«История»** в тулбаре
- Открывается полноэкранная панель

## Что показывает

- Список версий описания с временными метками
- Каждая запись — уникальная версия `description` (промежуточные изменения, где менялся только title или другие поля, пропускаются)
- Текущая версия помечена

## Сравнение версий

1. Выбери **базовую** версию (слева) — обычно более старую
2. Выбери **сравниваемую** версию (справа) — обычно более новую
3. Diff отображается построчно:
   - 🟢 зелёный — добавленные строки
   - 🔴 красный — удалённые строки
4. Доступен предпросмотр Markdown для каждой из выбранных версий

## Восстановление

- Кнопка **«Восстановить эту версию»** под каждой версией
- Текущую версию восстановить нельзя (кнопка заблокирована)
- При восстановлении создаётся новый снепшот в истории — предыдущее состояние не теряется

## Детали реализации

- Эндпоинт: `GET /api/documents/{id}/history`
- Используется `DISTINCT ON (description_version)` — возвращается по одной записи на каждую уникальную версию описания
- Diff на фронте строится библиотекой `diff` (построчное сравнение)
- Модал рендерится через React Portal в `document.body`
