## Сущности

### ChatMessageEntity
Таблица: `chat_message`. Реализует `Message` (Spring AI) и `Persistable<Long>`.

| Поле | Тип | Описание |
|---|---|---|
| `id` | long | PK. `isNew()` → `id == 0L` |
| `conversationId` | String | FK → chat_topic.conversation_id |
| `content` | String | Текст сообщения |
| `type` | MessageType | USER, ASSISTANT, SYSTEM, TOOL |
| `position` | long | Порядок в беседе |
| `summarized` | boolean | Попало ли в саммари |
| `summary` | boolean | Является ли само сообщение саммари |
| `createdAt` | LocalDateTime | Время создания |

Метод `getMessage()` возвращает Spring AI wrapper в зависимости от type:
- `USER` → `UserChatMessage`
- `ASSISTANT` → `AssistantChatMessage`
- `SYSTEM` → `SystemChatMessage`
- `TOOL` → `ToolChatMessage`

### ChatTopicEntity
Таблица: `chat_topic`. Реализует `Persistable<String>` (ID = conversationId).

| Поле | Тип | Описание |
|---|---|---|
| `conversationId` | String | PK, UUID |
| `user` | String | Имя пользователя |
| `isUser` | boolean | `true` = пользователь переименовал вручную, `false` = авто-имя |
| `topic` | String | Название чата |
| `createdAt` | LocalDateTime | `@CreatedDate` |
| `updatedAt` | LocalDateTime | `@LastModifiedDate` |
| `isNew` | boolean | `@Transient` |

**Приоритет переименования:** если `isUser = true`, TopicFunction не перезаписывает `topic`.

### Spring AI Message Wrappers
В пакете `model.chat.spring`:

| Класс | Назначение |
|---|---|
| `IMessage` | Интерфейс с методами `getContent()`, `getMessageType()`, `getMetadata()` |
| `UserChatMessage` | Оборачивает ChatMessageEntity с type=USER |
| `AssistantChatMessage` | Оборачивает ChatMessageEntity с type=ASSISTANT |
| `SystemChatMessage` | Оборачивает ChatMessageEntity с type=SYSTEM |
| `ToolChatMessage` | Оборачивает ChatMessageEntity с type=TOOL |
