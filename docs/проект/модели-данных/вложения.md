## Сущности

### AttachmentEntity
Таблица: `attachments`. Вложение принадлежит либо документу, либо чату.

| Поле | Тип | Описание |
|---|---|---|
| `id` | Long | PK |
| `ownerType` | String | Дискриминатор: `"document"` или `"chat"` |
| `documentId` | Long | FK → documents.id (когда ownerType = 'document') |
| `conversationId` | String | FK → chat_topic.conversation_id (когда ownerType = 'chat') |
| `fileName` | String | Оригинальное имя файла |
| `contentType` | String | MIME-тип, например `text/plain`, `text/markdown` |
| `fileSize` | long | Размер в байтах |
| `content` | String | Текстовое содержимое (null для бинарных — пока не поддерживаются) |
| `summary` | String | AI-саммаризация файла (null пока не запрошена) |
| `sourceUrl` | String | URL источника (Jira, Confluence), null для пользовательских загрузок |
| `createdAt` | OffsetDateTime | Время создания |
| `updatedAt` | OffsetDateTime | Время последнего изменения |

### AttachmentEmbeddingEntity
Таблица: `attachment_embeddings`. Один embedding на вложение (UNIQUE по attachmentId). Аналог DocumentEmbeddingEntity.

| Поле | Тип | Описание |
|---|---|---|
| `id` | Long | PK |
| `attachmentId` | Long | FK → attachments.id (UNIQUE) |
| `embedding` | float[] | Вектор pgvector |
| `model` | String | Модель эмбеддинга |
| `updatedAt` | OffsetDateTime | Время последнего обновления |
