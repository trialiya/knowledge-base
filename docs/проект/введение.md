# Введение

Добро пожаловать в базу знаний проекта **AI Chat Knowledge Base** — системы, объединяющей чат с искусственным интеллектом и структурированное хранилище документов.

## Что это за проект?

AI Chat Knowledge Base — это веб-приложение на Spring Boot и React, которое позволяет:

- **Общаться с AI** через чат-интерфейс с поддержкой Markdown, вложений и потоковой генерации ответов
- **Управлять базой знаний** — создавать, редактировать и организовывать документы в древовидной структуре
- **Искать информацию** — как по документам базы знаний, так и по файлам Git-репозитория
- **Работать с Git-репозиторием** — просматривать файлы, историю коммитов и диффы

## Ключевые возможности

### Чат
- Потоковая генерация ответов (SSE)
- Поддержка Markdown с подсветкой синтаксиса
- Загрузка вложений (изображения, файлы)
- Интеграция с базой знаний и Git-репозиторием через инструменты AI
- Сохранение истории чатов

### База знаний
- Древовидная структура документов и папок
- Markdown-редактор с живым предпросмотром
- Полнотекстовый и семантический поиск
- Экспорт документов в Markdown
- Вложения для документов
- История изменений с возможностью восстановления версий

### Git-интеграция
- Просмотр файлов репозитория
- История коммитов с диффами
- Поиск по содержимому файлов (grep)
- Структурный обзор файлов кода (outline)

## Технический стек

| Слой | Технологии |
|------|-------------|
| **Бэкенд** | Java 21, Spring Boot 3.x, Spring AI, Spring WebFlux |
| **Фронтенд** | React 19, React Router 7, CSS Modules |
| **База данных** | PostgreSQL (основная), H2 (для разработки) |
| **Поиск** | PostgreSQL full-text search + pgvector (семантический) |
| **Git** | JGit для работы с репозиторием |
| **Сборка** | Gradle (бэкенд), Vite (фронтенд) |
