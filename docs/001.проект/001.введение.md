# Введение

## О проекте

**Knowledge Base** — веб-приложение для управления иерархической базой знаний с AI-ассистентом. Позволяет хранить, организовывать и искать документы в формате Markdown, а также общаться с интеллектуальным чат-ботом, который отвечает на основе содержимого базы знаний и Git-репозитория проекта.

## Основные возможности

1. **Иерархическая база знаний** — папки и документы с древовидной структурой, пагинация, drag-free reorder
2. **Markdown-редактор** — создание и редактирование документов в интерфейсе
3. **Гибридный поиск** — комбинация keyword (trigram) + semantic (векторный, cosine similarity) с настраиваемыми весами
4. **AI-чат** — ассистент на базе OpenAI-совместимого API с инструментами для работы с документами, Git и вложениями
5. **Вложения** — загрузка текстовых файлов к документам и чатам, AI-суммаризация, векторный поиск по содержимому
6. **Git-интеграция** — просмотр дерева файлов, истории коммитов, diff'ов и контента файлов из репозитория
7. **Экспорт документов** — выгрузка базы знаний в файловую систему
8. **Atlassian-интеграция** — создание чатов с контекстом из Jira и Confluence
9. **Docker** — готовый compose-файл для быстрого развёртывания
