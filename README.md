# Knowledge Base

Веб-приложение для управления базой знаний с AI-ассистентом. Документы в Markdown, иерархическая структура, умный поиск и чат с ИИ.

## Что умеет

- 📁 **База знаний** — папки и документы с древовидной структурой
- ✏️ **Markdown-редактор** — создание и редактирование документов в интерфейсе
- 🔍 **Гибридный поиск** — keyword + смысловой (векторный) поиск
- 🤖 **AI-чат** — ассистент, который отвечает на основе содержимого базы знаний
- 📎 **Вложения** — загрузка файлов к документам и чатам
- 📤 **Экспорт** — выгрузка базы знаний в файловую систему
- 🐙 **Git-анализ** — чтение файлов, коммитов, диффов, структурный анализ кода
- 🔗 **Atlassian** — импорт из Jira и Confluence
- 🐳 **Docker** — готовый compose-файл для быстрого развёртывания

Подробнее — [Введение](docs/001.проект/001.введение.md)

## Быстрый старт

```bash
cd docker
cp example.env .env
# Укажите AI_API_KEY, AI_EMBED_API_KEY и PROJECT_PATH_MOUNT
docker compose up -d
```

Приложение будет доступно на `http://localhost:8080`.

Пошаговая инструкция и решение проблем — [Руководство по установке](docs/001.проект/008.руководство-по-установке.md)
